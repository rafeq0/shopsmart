# User Stories - ShopSmart Sprint 1

## SS-1: User Registration (feature/SS-1-register)

**As a** new customer,  
**I want to** register with my name and email,  
**so that** I can create a shopping account.

**Acceptance Criteria:**
- [ ] All fields are required (empty submission prevented)
- [ ] Success message appears after valid registration
- [ ] Registration data saved in localStorage

**Priority:** High  
**Assignee:** @rafeq0  
**Related commits:** #SS-1

---

## SS-2: Product Browsing (feature/SS-2-catalog)

**As a** registered customer,  
**I want to** browse a catalog of products displayed as cards,  
**so that** I can see what items are available.

**Acceptance Criteria:**
- [ ] Each product card shows name, price, and image
- [ ] Page is responsive (works on mobile/desktop)
- [ ] Products rendered from JavaScript array

**Priority:** High  
**Assignee:** @rafeq0  
**Related commits:** #SS-2

---

## SS-3: Shopping Cart (feature/SS-3-cart)

**As a** customer,  
**I want to** add products to a cart and see the cart count update,  
**so that** I can review my selected items before checkout.

**Acceptance Criteria:**
- [ ] "Add to cart" increases counter immediately
- [ ] Cart page displays all items and total price
- [ ] Cart data persists after page reload (localStorage)
- [ ] "Clear cart" button removes all items

**Priority:** High  
**Assignee:** @rafeq0  
**Related commits:** #SS-3
